Installation #back to top

  • Step 1 - Unzip the vCard - Digital Business Card Builder.zip which you have downloaded from the Envato Market.
  • Step 2 - Find the Code folder from the downloaded package and take out all the data from Code folder and upload to your server. Make sure to copy .htaccess file.
  • NOTE : - Please chmod application/config/database.php, install/data and all files inside this folder to 777.
  • Step 3 - Open your web browser and go to your domain url. For example: http://yourdomain.com Now you will be redirected to the installation page.
  • Step 4 - Enter your the Database Details, if not created databse then create a MySql Database via your web hosting providers Control Panel. Then, enter your Admin Email and Password which will be required to login in the application. This credentials will needed on everytime you login.
  • NOTE : - Password requires 8 characters minimum and no whitespace.
  • Step 5 - Now click on install and wait untill process finished. The system will create necessary tables in database, files and configuration, Then you will be automatically redirected to the login page.
  • NOTE : - If you get 404 OR 505 error then wait for 30 seconds and refresh the page. Again make sure .htaccess is uploaded.
  • NOTE : - vCard is now ready to run but we recommended you to first setup and customized vCard application according to your needs from settings section.

Custom Domain #back to top

  • You can manage custom domain from here here.
  • NOTE : - There is no guarantee custom domain will work because it's totally depends on you Server / Host. Support will not provided for this feature.
  • To activate this feature go to yourdomain.com/settings and enable option Turn off custom domain system
  • Once user / subscriber point domain to your server and make request for custom domain you have to add that domain to your Server / Host like normal domain.
  • Login to your server panels like cPanel and go to the Parked Domain OR Addon Domain depend on your panel then add custom domain of your user / subscriber.
  • Make sure to point custom domain to same directory as main domain.
  • For example, your main domain is 1domain.com connected with public_html/1domain directory. Now your user added custom domain 2domain.com. When you add this 2domain.com to your Server / Host it must have same directory as public_html/1domain. Means this both domain can access same content of one website.
  • Make sure to activate domain from admin panel after adding domain to your Server / Host.
