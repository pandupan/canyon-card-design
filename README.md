# 🏔️ Canyon Card Design

Website admin panel untuk manajemen **kartu desain** menggunakan PHP, CodeIgniter 4, dan MySQL. Terdapat 2 role pengguna (user dan admin).

## ✨ Fitur

- **Role Admin** — Manajemen data, CRUD, dan pengaturan sistem
- **Role User** — Akses fitur sesuai level pengguna
- **CodeIgniter 4** — PHP framework ringan dan cepat
- **MySQL** — Database relasional

## 🛠️ Tech Stack

- **PHP** — Backend language
- **CodeIgniter 4** — PHP framework
- **MySQL** — Database
- **Bootstrap** — Frontend UI

## 🚀 Cara Menjalankan

```bash
# Clone dan atur konfigurasi database
cp .env.example .env
# Edit .env untuk koneksi database
php spark serve
```
